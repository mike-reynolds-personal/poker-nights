Poker Rooms Now

Private Texas Hold'em poker rooms for free!

Poker Rooms Now is a new Texas Hold'em poker site which was created with a few simple goals in mind;

  • Easy hosting and quick set-up. Enable anyone to host and play a private game of Texas Hold'em with friends, with the minimal amount of set-up required and no need to download or install any additional software.
  • Only focus on private games. We are not a casino or gambling web site - we provide our Software as a Service for you to play the online equivalent of a private home game.
  • Transparent pricing. There are No rake fees! All games have a fixed hosting fee for up to 12 hours play (post-launch).
  • Keep it social. A good home game is as much about the social as it is about the poker, therefore there are no timers to force you in to action or distract you from a conversation.. As we don't charge a rake-fee, we don't mind how many rounds you play... 5 or 500, leaving more time for banter.
  • Keep it fun. There are no big-money pay-outs and no professional players in your game (unless you invite them), so you can be assured that no one is there just to earn money from you.

How to host your own Game

Do you want to play Texas Hold'em with friends for free on a private table?

Our site is currently 'pre-launch', therefore in exchange for constructive feedback all games are free to create and play.

It's easy to play, and easy to start playing:

  • Sign-in or create an account using the 'Sign-In' button in the top right-hand corner.
  • Once signed-in, you can Create a Game to schedule a private game. Games can be available immediately, but are archived 24 hours after the scheduled start time. A game is available 15 minutes prior to the scheduled time.
  • Choose your options, such as initial blinds, game variant and buy-in amount, then click 'Create'.
  • You're provided a unique game link which you can save and share with up to 8 friends (plus host). If you provide participant emails, they're sent and email with a calendar appointment which contains all of the relevant details.
  • If you schedule a game less than 15 minutes in the future, feel free to click the game link straight away to check out the table.
  • Around the scheduled start time, use the link to go to the table and choose a seat. Enter a name along with the amount you want to place in your game wallet. We suggest you enter roughly the same amount you would usually take to a home game.
  • Once more than the minimum required players (two for Texas Hold'em) are seated, the host can start the game. Note that only the player with an email address matching the Host Email entered in the Create Game form can start the first game.
  • Instructions on using the table, along with the hand ranks, are available from the in-game menu. The host has additional options to help them control the game, but they no visibility of any cards apart from their own, so they play along with everyone else.

Current development and release status

Poker Rooms Now is in a 'pre-launch' status. This means that the software is free to use in it's current form and whilst we endeavour to ensure it works as expected, it is offered without warranty or acceptance of any liability.

If you encounter any issues then please email Support with a description of the issue and how it occurred, and we'll work hard to address it.

Along with getting all of the basics in place, we have some exciting developments in the pipeline, including the ability to transfer winnings between players and run tournament games. If you have any suggestions then please email us

Known issues:

  • Game wallet is not saved between games
  • Games running longer than four hours can disconnect players due to session timeouts
  • The account activation email can redirect you to the wrong page
  • British pounds (GBP) is the only supported currency
A full list of the updates/ improvements is available in the Instructions dialog available from the game table.
